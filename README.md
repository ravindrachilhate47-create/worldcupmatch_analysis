# 🏏 World Cup Data Analysis

Welcome to the **World Cup Data Analysis** project! This repository contains a comprehensive
data analysis of World Cup matches, exploring insights, trends, and team performances.

---

## 📁 Repository Structure

* **`world_cup.ipynb`:** The main Jupyter Notebook containing all the Python code, data cleaning, visualizations, and insights.
* **`match_result.csv`:** The core dataset containing match results and statistics used for this analysis.

---

## 🛠️ Tech Stack & Libraries Used

* **Language:** Python 🐍
* **Libraries:** Pandas
---
